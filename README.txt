///////Welcome to goWar!///////////
----------------------------------

To play go War at least 3 computers are needed. 1 server and at least 2 clients. There is no max player limit

Server will intialize all clients as red or blue teams. The winner team with the last computer standing wins.
To play, one user will send a start command to all players. On the start of war clients will choose an
enemy to start attacking. 1 spacebar hit = 1 damage point. Users die when they take 100 damage.
When only players on one team are remaining that team will be declared victorious.

Have fun!
